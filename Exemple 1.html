Javascript Python
Explication : Les fonctions en Javascipt et Python se définissent par deux manières différentes :

- Par l'instruction "Function" en Javascript

- Par l'instruction "def" en Python

Elles ont pour point commun la structure de la fonction, les deux langages ont dans la signature de la fonction : le nom de la fonction et les paramètres (exemple : def nom_de_la_fonction(paramètre)).

Les variables sont définis différement, en javascript on utilise l'instuction "var" précédant le nom de la variable.

Alors qu'en Python, il suffit juste de donner un nom de variable suivi d'un "=" (exemple: a= "exemple". Donc la variable a à pour valeur la chaine de caractère "exemple").

La fonction sert ici a voir si le chiffre donner par l'utilisateur, est un nombre pair ou impair. Pour cela :

-On divise le nombre donner par 2 (le "%" signifie diviser)

-Si le reste de cette division est égal à 0 (écris avec "=="), la fonction renvoie la chaine de caractère "pair"

-Sinon elle renvoie la chaîne de caractère "impair"

On appelle ca (if, else, elif) une structure conditionnel
